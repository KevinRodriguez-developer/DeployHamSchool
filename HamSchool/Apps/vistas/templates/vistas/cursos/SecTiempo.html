El tiempo PASADO SIMPLE lo utilizamos para referirnos a momentos que ya pasaron y se estructura de la siguiente manera.
[Sujeto] + [Verbo en pasado] + complemento
I + played + with the red ball
Yo + jugué + con la pelota roja ayer
[Sujeto] + [did not] + [Verbo base] + complemento
I + did not + play + with the red ball yesterday
Yo + no + jugué + con la pelota roja ayer
[Did] + [sujeto]+ [verbo base] + complemento ?
Did + she + play + with the red ball yesterday ?
ella + jugo + con la pelota roja ayer?
Yes + [sujeto] + [did]
Si, ella jugo con la pelota roja ayer
No + [sujeto] + [didn´t]
No, ella no jugo con la pelota roja ayer
El tiempo PRESENTE SIMPLE lo utilizamos para referirnos a las acciones que suceden en el momento y se estructura de la siguiente manera.
[Sujeto] + [verbo base] + complemento
I + talk + to my sister every night
Yo + platico + con mi hermana todas las noches
RECUERDA esta estructura es para WE, I, THEY, YOU
[Sujeto] + [do not] [verbo base] + complemento
I + [do not] + talk + to my sister every night
Yo + no platico + con mi hermana todas las noches
RECUERDA esta estructura es para WE, I, THEY, YOU
[DO] + [sujeto] [verbo base] + complemento ?
Do + [you] + talk + to your sister every night?
Yes + sujeto + verbo base / No + sujeto + [do not] + verbo base
Yes + I + talk + to my sister every night
No + I + do not + talk + to my sister every night
RECUERDA esta estructura es para WE, I, THEY, YOU
[Sujeto] + [verbo (ies/s/es)] + complemento
She + studies + every day
Ella + estudia + todo el dia
He + works + every Sunday
El + trabaja + todo los domingos
He + fixes + his car
El + arregla + su carro
RECUERDA esta estructura es para she, he, it
[Sujeto] + [does not] + [verbo base] + complemento
She + [does not ] + study + every day
Ella + no estudia + todo el dia
RECUERDA esta estructura es para she, he, it
[DOES] + [sujeto] [verbo (s,es,ies)] + complemento ?
Does + [she] + studies + every day?
Yes + sujeto + verbo (s, es, ies) / No + sujeto + [does not] + verbo (s, es, ies)
Yes + she + studies + every day
No + she + does not + study + every day
RECUERDA esta estructura es para she, he, it
